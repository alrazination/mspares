📋 Google API Setup Guide

  1. Go to console.cloud.google.com
  2. New Project → name it → Create
  3. APIs & Services → Library → Enable Google Sheets API + Google Drive API
  4. OAuth consent screen → External → fill name → Add yourself as Test User
  5. Credentials → + Create → OAuth Client ID → Web application
  6. Add your GitHub Pages URL to Authorised JS Origins
  7. Copy Client ID → paste above → Connect
